## 📝 고객 요구사항 정리
1. 신용카드 등록이 가능해야 합니다. 이 기능은 중요합니다.
2. 카드 정보에 대한 보안은 철저하게 유지되야 하며 카드번호나 보안코드는 입력시 보이지 않게 처리해야 합니다.
3. 여러카드 등록이 가능해야 합니다.
4. 테스트 URL이 필요합니다.

## 📋 기능 목록
- 카드
  - `<CardNumberInput />`
  - 만료일 등록
  - 소유자 입력
  - 카드 비밀번호 입력
- 구매 버튼 기능 
- 여러 카드 등록
- 보안 코드 등록
- 테스트 URL
